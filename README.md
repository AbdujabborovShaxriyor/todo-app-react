## 📝 Todo App React

A simple and modern Todo app built with React.  
Features include adding, editing, deleting, and marking tasks as completed. Designed to be a clean, minimal, and responsive UI.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Responsive and modern UI
- Local state management with React useState

## Tech Stack
- React
- CSS (modern styles with hover, shadow effects)
- JavaScript (ES6+)

## Live Demo
[Try the app online](https://todo-app-react-gamma-weld.vercel.app/)

## Installation

1. Clone the repo:
  git clone https://github.com/AbdujabborovShaxriyor/todo-app-react.git 

2. Install dependencies:
  npm install

3. Run the app:
  npm start

## Usage
- Type a task in the input box
- Click "Add task" to add
- Check the box to mark as completed
- Click "Edit" to change task text
- Click "Delete" to remove a task
